# 本地目标完成度（先不讨论收费）

更新时间：2026-01-03

## 本地目标定义（收费前的产品基线）

在决定广告/买断/订阅之前，先把下面这套“可长期运营的体验基线”打稳：

1) 核心查字/发音/笔顺体验稳定（不卡顿、不崩溃、可解释的失败态）
2) 字典可用且能联动练习（搜索/收藏/历史/一键练）
3) 每日一字闭环（首页入口 + streak + Widget + 每日提醒）
4) 设置与新手体验完整（主题/暗黑、音量安全、首次引导、语言、帮助/隐私）

## 进度概览

- **待完成（P0/P1，建议收费前完成）：4 项**
- **可选增强（P2/长期）：3+ 项**
